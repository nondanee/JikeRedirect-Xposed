# JikeRedirect-Xposed

~~一个护眼 Xposed 模块~~ 即刻回来啦! (2020.6.10)

## Feature

- 连接到 Jellow API HOST

- 移除 Webview 的外部链接重定向 (`redirect.jellow.club` 用不了)

- 强制显示招呼和通知 (聊天页 503 错误)

- 跳过引导页 (引导页显示不了)

- 修改圈子动态 API (Jellow 圈子 API 有改动)

- 修正 CDN 错误资源
